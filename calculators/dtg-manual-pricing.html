About Manual DTG Pricing

How Manual DTG Pricing Works

This calculator helps you price Direct-to-Garment printing services for customer-supplied garments. Enter your base garment cost, select a print location, and specify the quantity to get accurate pricing.

The calculator uses the same pricing engine as our standard DTG pricing, with print costs and margins from our pricing database. Orders under 24 pieces include a $50 Less Than Minimum (LTM) fee, which is divided across all items to cover setup and production costs.

Detailed Pricing Explanation

DTG Manual Pricing Calculator Guide

Use this guide to manually calculate DTG prices with a calculator. All formulas match the automated system exactly.

Quick Reference: Common Print Costs (24-47 tier)
Location Size Print Cost
Left Chest / Back of Neck 4"4" $6.00
Full Front / Full Back 12"16" $10.00
Jumbo Front / Jumbo Back 16"20" $12.00

Note: Print costs decrease at higher tiers (48-71 and 72+). Margin is always 0.6 for DTG.

Single Location Pricing Formula

Step-by-Step Calculation:

  1. Garment with Margin = Garment Cost 0.6
  2. Add Print Cost = Result from Step 1 + Print Cost (from table above)
  3. Round Up = Round result to next half-dollar ($.50)

Example: 30 shirts, $5.00 garment cost, Left Chest print

  1. $5.00 0.6 = $8.33
  2. $8.33 + $6.00 = $14.33
  3. $14.33 rounds UP to $14.50 per shirt

Total Order: $14.50 30 = $435.00

Combo Location Pricing Formula (2 Locations)
Important: Combo pricing uses double rounding - this is different from single locations!

Step-by-Step Calculation:

  1. Calculate First Location = Use single location formula (see above)
  2. Add Second Print Cost = Add ONLY the print cost (no margin) for second location
  3. Round Up Again = Round result to next half-dollar ($.50)

Example: 30 shirts, $5.00 garment cost, Left Chest + Full Back

  1. First location (LC):
    • $5.00 0.6 = $8.33
    • $8.33 + $6.00 = $14.33
    • Rounds to $14.50
  2. Add second print cost: $14.50 + $10.00 = $24.50
  3. $24.50 is already on half-dollar, so $24.50 per shirt

Total Order: $24.50 30 = $735.00

Small Order Fee (Less Than 24 Pieces)

How the $50 Small Batch Fee Works:

  • Orders under 24 pieces have a $50 flat fee added to the total order
  • This fee is NOT included in the per-shirt price
  • The per-shirt price uses 24-47 tier rates

Example: 10 shirts, $8.00 garment cost, Left Chest

  1. Calculate unit price using 24-47 tier:
    • $8.00 0.6 = $13.33
    • $13.33 + $6.00 = $19.33
    • Rounds to $19.50 per shirt
  2. Subtotal: $19.50 10 = $195.00
  3. Add small batch fee: $195.00 + $50.00 = $245.00 total
Rounding Rules (Always Round UP)

All prices round UP to the next half-dollar ($0.50):

Calculated Price Rounded Price
$14.01 to $14.50 $14.50
$14.51 to $15.00 $15.00
$15.01 to $15.50 $15.50

Calculator tip: Multiply by 2, round up to next whole number, then divide by 2.
Example: $14.33 2 = 28.66 rounds to 29 29 2 = $14.50
