Shopworks: Embroidery Order Type

📋 New Embroidery Order Process

Simplified system for all embroidered items

✨ Effective: August 15, 2025

Important Change!

The "Caps" order type has been REMOVED from Shopworks.
All embroidered items now use the "Embroidery" order type only.

1

One Order Type for All Embroidery

All embroidered items—whether caps, garments, or other items—are now written as an "Embroidery" order type in Shopworks.

What this means:
  • ✅ Caps → Use "Embroidery"
  • ✅ Jackets → Use "Embroidery"
  • ✅ Polos → Use "Embroidery"
  • ✅ Beanies → Use "Embroidery"
  • ✅ Any embroidered item → Use "Embroidery"
2

Critical: Price Caps and Garments Separately

Even though caps and garments are on the same order, they MUST be priced separately based on their individual quantities.

Why? Caps and garments run on different embroidery machines, so they cannot be combined for quantity pricing.

❌ Wrong Way

  • 12 Caps + 12 Jackets = 24 pieces total

Incorrect Pricing:

Using 24-piece price break ❌ WRONG

✅ Right Way

  • 12 Caps (separate group)
  • 12 Jackets (separate group)

Correct Pricing:

Caps: 12-piece pricing
Jackets: 12-piece pricing

🎩 Special Rule: Beanies are "Flats"

Important Exception!

Beanies are embroidered on our flat machines, not cap drivers.
This means beanies are priced as "Flats" (regular Embroidery).

💡 You CAN combine beanies with other flat embroidery items for quantity price breaks!

🔍 Quick Reference Guide

Item Type Order Type Machine Type Can Combine for Pricing?
Caps Embroidery Cap Driver No - Price separately
Beanies Embroidery Flat Machine Yes - With other flats
Jackets Embroidery Flat Machine Yes - With other flats
Polos Embroidery Flat Machine Yes - With other flats
T-Shirts Embroidery Flat Machine Yes - With other flats

📌 Example Scenario

Order: 12 caps + 12 beanies + 12 jackets

Caps: 12-piece price (separate) Beanies + Jackets: 24-piece price (combined)
