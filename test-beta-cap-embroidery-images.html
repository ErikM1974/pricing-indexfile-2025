What to Check

  1. Product Image Gallery:
    • Main product image should load and display correctly
    • Image should change when different colors are selected
    • Thumbnails should appear if multiple views are available
    • Zoom functionality should work on hover/click
  2. Color Swatches:
    • All available colors should display as swatches
    • Color names should appear below swatches
    • Selected color should be highlighted with active state
    • Clicking a swatch should update the main image
    • Selected color display should update above the swatches
  3. Integration:
    • URL should update when color is changed
    • Pricing should remain functional
    • No console errors should appear

Console Output

Open the browser console (F12) to check for any errors or warnings when testing the beta page.
