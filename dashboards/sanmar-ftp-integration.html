Download SanMar files

These are the data files SanMar publishes on their FTP server. Click Download to save one to your computer — no FileZilla needed — then import it into Caspio. The product & pricing master is SanMarPI-Bulk; it feeds garment CASE_PRICE into every quote builder. SanMar pricing changes only every few months, so grab a fresh copy when it does.

After you download — import into Caspio

  1. Caspio app Sanmar Pricing 2026 → table Sanmar_Bulk_251816_Feb2024 → Import the CSV you just downloaded.
  2. Import mode: Update & add records, match on UNIQUE_KEY.
  3. Leave unmapped so your own values are kept: IsTopSeller, IsNew, Display_Image_URL (formula).
  4. Done — this one table feeds every quote builder, so all pricing surfaces refresh within ~15 min (Customer Catalog, Quick Quote, and the DTG/DTF/EMB/SCP builders).

Prefer a one-click desktop tool?

There's also Download-SanMar-Pricing.bat in the sanmar-ftp-box folder — double-click it to pull the newest pricing file straight to your Downloads folder without opening this page. Same file, same next step (import into Caspio).

One-time setup

The download buttons need the SanMar FTP password stored on the app as the config var SANMAR_FTP_PASSWORD (with optional SANMAR_FTP_USER, default 6920). Once it's set, this page works for any admin — no local FileZilla or password prompts.
