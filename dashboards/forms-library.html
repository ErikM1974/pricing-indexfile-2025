Adding a form

Add a row to the Forms_Library table in Caspio (name, category, PDF_URL, optional Fill_Online_URL, Is_Active = Yes) and it appears here automatically — no deploy. The PDF itself lives in the site's /forms/ folder (ask Erik/Claude to add it) or any hosted URL.
