My Recommendation: Option 3 - Header Dropdown

Why it's the best choice:

  • Least Intrusive: No floating buttons, no slide-out panels, no modals covering content
  • Familiar Pattern: Users understand shopping cart dropdowns
  • Always Accessible: Quote button in header is always visible but not annoying
  • Mobile-First: Works perfectly on all devices
  • Fast Access: One click to see quote summary, two clicks for full details

Implementation is super simple:

  1. Add quote button to existing header
  2. Simple dropdown on click/hover
  3. Auto-close when clicking outside
  4. Full quote details open in a proper page/modal only when needed

User Flow:

Click Quote (3) ‚Üí See dropdown summary ‚Üí Click "View Full Quote" ‚Üí Opens dedicated quote page
