Art File Definitions

Picture Files

.PNG (Portable Network Graphic)

Used for website development. Transparent Background

.JPG (Joint Photographic Group)

Used for digital images across the internet and social media

.PSD (Photoshop Document)

File created in Adobe Photoshop with layers. All Layers are editable and can be saved in layers or combined into a single image.

Vector Files

.AI (Adobe Illustrator) Vector

File created in Adobe Illustrator with layers. All Layers are editable and can be saved in layers or combined into a single image.

.CDR (Coreldraw) Vector

File created in Coreldraw with layers. All Layers are editable and can be saved in layers or combined into a single image.

.PDF (Portable Document File)

Used for sending document and text formatting independent of proprietary software applications. If saved and sent as editable, it is Vector art.

.EPS (Encapsulated Postscript)

.EPS files are an old standard that was used in print. It is Vector based and will work for the most part but not an ideal Vector file.
